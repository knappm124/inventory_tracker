# Inventory Tracker
authored by Melissa Knapp

## Objective

The goal of this project is to create a platform to track inventory. The tracker will contain fields for location, status, name, price, and "tags". Tags will be broken into 2 parts, the first is the title of the tag, like Color. The second part is the options to choose from, like "Red", "Green", or "Blue". This project is being developed for organizing Pysanky Eggs, but the goal end state can be used for any type of inventory.

## Minimum Viable Product

The MVP end state is a program that has statuses, locations, and tags pre-populated. Users will only be able to add, edit, and remove inventory items.

## Goal End State

Goal end state includes the ability to edit tags, status options, and locations, along with an amount of inventory for each item and a photo for each item. It will also involve remote database for storing user data and a mobile platform.
